PHP - CLI

PHP peut être utilisé en ligne de commande afin de créer des scripts shell ; il est possible d'exploiter un script PHP sans serveur web (Apache, Ngenix,...)

Serveur web

Un serveur web de développement est disponible avec PHP, les requêtes sont servies depuis le dossier de travail courant où PHP a été démarré.


php -S localhost:8000

Point d'entrée

Créez avec votre éditeur de code un fichier nommé index.php à la racine de votre répertoire web.

index.php va permettre le chargement de l'ensemble de vos scripts pour afficher votre HTML.
